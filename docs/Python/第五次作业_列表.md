**7-2 大于身高的平均值**

```
n=input().split()
sum=0
stu=0
for x in n:
    x=int(x)
    sum+=x
    stu+=1
if stu!=0:
    aug=sum/stu
for x in n:
    x=int(x)
    if x>aug:
        print(x,end=" ")
```

7-5 班级排队

```
stu_num=int(input())
boy=[]
girl=[]
high=input().split()
male=input().split()
for x in range(0,stu_num):
    if male[x]=="0":
        boy.append(int(high[x]))
    elif male[x]=="1":
        girl.append(int(high[x]))
boy.sort()
girl.sort()
stu=girl+boy
stu__last=stu.pop(-1)
for y in stu:
    print(y,end=" ")
print(stu__last)
```
