##### 7-6 判断字符串头尾结构

```
a=input()
flag=True
low=0
high=len(a)-1
while low<high:
    if a[low]!=a[high]:
        flag=False
        break
    low+=1
    high-=1
if flag:
    print("yes")
else:
    print("no")
```

##### 7-12 字符串判等

```

sp_a=a.split()
join_a="".join(sp_a)
low_a=join_a.lower()
sp_b=b.split()
join_b="".join(sp_b)
low_b=join_b.lower()
if low_a==low_b:
    print("YES")
else:
    print("NO")
```
