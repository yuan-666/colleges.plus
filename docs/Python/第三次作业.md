5.打印特殊三角形

输入一个数字n，输出一个n层的特定三角形，三角形内数字增长是有规律的。

```
n=int(input())
sum=1
num=1
for a in range(n):
    for b in range(n,a,-1):
        if b==a+1:
            print(sum)
        else:  
            print("{:<3}".format(sum),end="")
        sum += b
        b += n
    num+=1
    sum=num
```

7.求满足条件的斐波那契数

斐波那契数，亦称之为斐波那契数列，指的是这样一个数列：1、1、2、3、5、8、13、21、……，这个数列从第3项开始，每一项都等于前两项之和。求大于输入数的最小斐波那契数。

```
n=int(input())
sum1=1
sum2=1
sum3=0
while n>sum3:
    sum3=sum1+sum2
    sum1=sum2
    sum2=sum3
print(sum3)
```



8.统计成绩

输入一批学生成绩，计算平均成绩，并统计不及格学生人数。

```
sum = 0
no_pass = 0
stu = 0
while True:
    n = input()
    if (n<'0' or n>'9') and stu==0:
        print("没有学生")
        break
    else:
        n=eval(n)
        if n<=0:
            print("平均分={:.2f},不及格人数={}".format(sum / stu, no_pass))
            break
        else:
            if n < 60:
                no_pass += 1
                sum += n
            else:
                sum += n
            stu += 1
```

10.s=1+2+3+...+n直到s>=m

s=1+2+3+...+n，当s刚超过某个数时，停止运算。求n和s的值

```
n=int(input())
s=0
s1=1
sum=0
while s<=n:
    s=s1+s
    s1+=1
    sum+=1
print("{}=1+2+...+{}".format(s,sum))
```
